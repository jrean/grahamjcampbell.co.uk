Laravel Framework

I have been contributing to the Laravel Framework for just over a year and am the second highest contributor, Taylor Otwell being the first. On 24th September 2014, Taylor made me part of the official Laravel organisation on GitHub. My contributions include adding new features, fixing bugs, and reviewing other people’s pull requests and issues. Laravel is an enormously popular PHP framework, with 2.5 million downloads through Composer so far.

The League of Extraordinary Packages

I am co-maintainer of Factory Muffin (previously named Factory Muff) alongside Scott Robertson, and have spent a lot of time writing documentation as well as rewriting/restructuring the code twice for 2.0 and 3.0 releases (3.0 is unreleased at the time of writing). The goal of this package is to enable the rapid creation of objects for the purpose of testing. I am not an official maintainer of Flysystem, however my contributions to that project are significant, and include many bug fixes, and improvements.

Class Preloader for PHP

I co-maintain this repo alongside Michael Dowling (the creator of Guzzle), and we have very recently hit the 2 million downloads mark. My contributions to this repo include a code cleanup, upgrading to the new PHP Parser library, and adding a test suite. Class Preloader optimizes class loading performance by generating a single PHP file containing all of the autoloaded files, and is also shipped with the Laravel core.

Links:

  • https://github.com/mtdowling/ClassPreloader
  • https://packagist.org/packages/classpreloader/classpreloader

Bootstrap CMS

I am creator and project leader of Bootstrap CMS. Bootstrap CMS is a PHP CMS powered by Laravel 4.2 and Sentry 2.1. It utilises many of my packages including Laravel Core, Laravel Credentials, and Laravel Auto Presenter.

Links:

Laravel Auto Presenter

I co-maintain “Laravel Auto Presenter” with Shawn McCool. This package automatically decorates objects bound to views during the view render process.

Other Laravel Packages

I currently maintain 21 Laravel packages by myself, and the combined number of downloads of these packages is approaching 100,000. All my packages follow PSR-2 and PSR-4, following semantic versioning, and are available on Packagist for download through Composer. Most of my packages have complete test suites and are well documented. A lot of my code is generic and I try to keep code coupled to Laravel within the service providers where possible.
